# 終活TODO アプリ

「もしも」の備えを整えるための、**ログイン不要・サーバー非保持の静的ブラウザツール**。
台帳の下書き作成、カード明細からの定期課金検出、公開版/完全版の書き出し（CSV/PDF）、
完全版の物理保管・引き継ぎ手順の表示までを、すべてブラウザ内で完結します。

> 設計原則: このアプリは**足場 (scaffolding)** であって構造体ではありません。
> 明日このアプリが落ちても、ユーザーの備え（印刷した紙＋物理保管）は一切壊れません。
> 詳細は [`BUILD_INSTRUCTION.md`](./BUILD_INSTRUCTION.md) を参照。

## 保管・開示モデル

秘匿情報（完全版）は **印刷して物理保管**（家庭用金庫・鍵付き引き出し＋封筒・銀行の貸金庫）します。
アカウント登録は不要です。いざという時の引き継ぎ（開示）は、**保管場所と開け方を平時に家族へ伝えておく**
ことで担います（中身そのものはメッセージ等に流さない）。公開版（地図）はデジタルで家族と共有します。

> 当初設計は Bitwarden の緊急アクセスに開示を委譲していましたが、家族全員のアカウント登録が
> 採用の障壁になるため、物理保管モデルに変更しています。Bitwarden への依存はありません。

## ハードルール（実装が守っていること）

- バックエンドにユーザーデータを保存しない（サーバーDB・永続ストレージなし）
- 認証機能を持たない（ログイン・セッション・トークンなし）
- 外部APIにユーザーデータを送らない（LLM不使用。明細解析もすべてブラウザ内）
- 開示（引き継ぎ）はアプリで実装せず、物理保管＋家族への伝達に委譲（手順表示のみ）
- 正本はアプリの外（印刷した紙の完全版 ＋ 物理保管）。IndexedDB は揮発しうる下書き

## 技術スタック

- Next.js（Pages Router）+ TypeScript / 静的エクスポート（`output: 'export'`）
- Tailwind CSS
- IndexedDB（`idb`、下書き専用）
- PDF生成: `pdf-lib` + `@pdf-lib/fontkit`（Noto Sans JP を `public/fonts/` に同梱して埋め込み）
- CSVパース: `papaparse` / CSV生成は自前（依存なし、BOM付きでExcel対応）

## 画面

| ルート             | 役割                                                         |
| ------------------ | ------------------------------------------------------------ |
| `/`                | 開始フロー（数問）→ 管理者向けTODOシート生成                 |
| `/todo`            | 管理者向けTODO（順序固定。自分を完走→家族に依頼）           |
| `/ledger`          | 台帳エディタ（型ごとの追加・編集、フィールド単位の公開/非公開）|

| `/storage-guide`   | 完全版の印刷・物理保管・引き継ぎ手順の表示のみ               |
| `/family`          | 家族向けの簡易フロー（同じ静的URLの1ルート。認証なし）      |

## 開発

```bash
npm install
npm run dev      # http://localhost:3000
npm run build    # 静的エクスポート → out/
```

`npm run build` で `out/` に静的サイトが生成されます。Vercel 等にそのままデプロイできます
（サーバー実行経路・API Routes は持ちません）。

## データモデル（要点）

- 台帳 = レコードの集合。各レコードは型（銀行口座／クレカ／サブスク／保険 等）を持つ。
- 分類軸は**公開/非公開の1本のみ**。各フィールドに既定可視性があり、ユーザーが上書きできる。
- 公開版エクスポート = 実効可視性が `public` のフィールドのみ（家族と共有する“地図”）。
- 完全版エクスポート = 全フィールド（印刷して物理保管する本体）。
- 再編集の正本は**CSV**。`/export` の「CSVを読み戻す」で読み込み、編集を再開できる。

## フォントについて

PDF の日本語表示には Noto Sans JP（OFL）を `public/fonts/NotoSansJP-Regular.ttf` に同梱し、
`pdf-lib` でサブセット埋め込みしています。標準フォントは日本語を出力できません。
